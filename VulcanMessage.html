Class: VulcanMessage

VulcanMessage(type, appId, appVersion)

VulcanMessage Message type for sending messages between host applications. A message of this type can be sent to the designated destination when appId and appVersion are provided and valid. Otherwise, the message is broadcast to all running Vulcan-enabled applications. To send a message between extensions running within one application, use the CSEvent type in CSInterface.js.
